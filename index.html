El proyecto tiene como propósito desarrollar un gestor documental flexible y moderno, usando las nuevas tecnologías de la información, las mejores prácticas y estándares disponibles en la industria, enfocándose en la gestión de documentos de las empresas, generando un cambio sobre la producción de documentos físicos.

INTRODUCCIÓN

En el marco de nuestro proyecto “ADOC Tú Gestor Documental” haremos uso de Figma para diseñar los mockUps de identidad de nuestro gestor, colores, tipo de letra, logo, botones, etc. ReactJS como lenguaje para el frontend, Laravel como framework de php será el responsable de la lógica de negocio y MariaDB será la base de datos donde persistirá la información y correlación de roles de los usuarios de nuestra plataforma.

Descripción del Proyecto y Entregables

El proyecto tendrá tres desarrollos, uno para implementarse en WEB, otro para Mobile y otro para Desktop.
Se tendrá en dos partes cada implementación una Frontend y otro Backend. Los entregables se harán siguiente la metodología Scrum, que es por sprints.

OBJETIVOS ESPECIFICOS

- Gestionar ciclo de vida de los documentos.
- Diseno de los MockUps.
- Conexion con la Base de datos.
- Generar alertas basadas en la uso de metodologia Scrum.
- Uso de herramientas de versionamiento de codigo.
- Relacionar documentos basados en su informacion.

Base de datos

La encriptación SHA256 se implemento sobre los datos de la base de datos.
La encriptación SHA256 es una medida de seguridad fundamental para proteger las bases de datos que contienen información sensible. Su implementación ofrece una sólida protección contra accesos no autorizados, robo de datos y otras amenazas.
